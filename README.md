# Images Events Store

An elegant black and white themed React application for uploading reference images to find photos from events.

## Features

- Clean, minimalist black and white design
- Fully responsive layout (mobile, tablet, desktop)
- Drag and drop image upload
- Image preview
- Email notification system
- Elegant success confirmation

## Getting Started

### Installation

```bash
npm install
```

### Development

```bash
npm run dev
```

### Build

```bash
npm run build
```

### Preview Production Build

```bash
npm run preview
```

## Tech Stack

- React 18
- Vite
- CSS3 (Responsive Design)
